dalek-status
==========

![initial](initial.png)

Overview
--------

An application than provides a visualization of the current sensory information as recieved from the Webtrends Dalek.


Requirements
------------

nodejs >= 0.10


Install
-------

```
npm install
```

Running
-------
```
./run.sh
```
